# chat_app_ttcs

Flutter project.

## Các folder trong project

- db: chứa các class xử lý tương tác với CSDL. (controller)
- forms: chứa các class trả về widget thành phần tạo nên screen. (view)
- models: chứa các class đóng gói các đối tượng. (model)
- process: chứa các class để xử lý tên,email, tạo mật khẩu ngẫu nhiên. (utils)
- screens: chứa các màn hình chính. (view)
